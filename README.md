## Overview

**SILVER7** is a dark-themed store management system built for retail businesses. It covers sales tracking, inventory, employee management, and logistics in one place.

> ⚠️ Currently in active development — no backend connected yet.

## Screens

| Screen | Description |
|--------|-------------|
| **Home** | Navigation hub with links to all modules |
| **Dashboard** | KPI cards (Sales, Orders, Employees, Products) + Revenue & Profit chart |
| **Sales** | Today / Weekly / Monthly / Total revenue + recent orders feed |
| **Order Detail** | Per-order breakdown — customer, payment, date, notes |
| **Inventory** | Product list with SKU, stock levels, low stock & out-of-stock alerts |
| **Employees** | Staff directory with role and status (Active / On Leave / Inactive) |
| **Logistics** | Shipment tracking with J&T Express integration |
